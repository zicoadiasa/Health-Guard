# HealthGuard v2.0

HealthGuard is a web-based health management application designed to help users monitor their lifestyle, nutrition, physical activity, blood sugar, and weight through AI-assisted recommendations.

## Features

- User Health Profile
- Personalized Goals
- Meal Planning
- Food Logging
- Activity Tracking
- Blood Sugar Monitoring
- Weight Tracking
- AI Insights
- Weekly Reports
- Reminder & Notification System

## Tech Stack

- Next.js
- TypeScript
- PostgreSQL
- Supabase
- Tailwind CSS

## Project Structure

```
src/
supabase/
public/
```

## Getting Started

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
